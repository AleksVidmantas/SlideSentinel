Information about the map

The third field on the nav bar is a sheet selector. This menu is populated with all the sheet names from the given google spread sheet. Selecting a sheet from this list will change which sheet the client loads data from. The default sheet is 'data' for testing purposes(this can be changed). The visualizer will search for two columns, one named longitude and one latitude and then represent those points on a map.

Each point will be visualized with a marker on the map. Additionally each marker can be clicked on to display more information from that marker. The information will be displayed in a table that will appear upon clicking.
